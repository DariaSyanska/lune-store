# 🌙 Lune Fashion Store

A modern minimalist fashion e-commerce website built with HTML, CSS, and JavaScript.

## ✨ Overview

**Lune** is a concept online store focused on elegant aesthetics, smooth interactions, and a premium user experience.
The project demonstrates a fully interactive front-end with product browsing, cart functionality, and a refined checkout flow.

## 🛠 Tech Stack

* HTML5
* CSS3 (custom design system)
* JavaScript (Vanilla JS)
* GSAP (animations)
* Lenis (smooth scrolling)

## 🚀 Features

* 🌗 Light / Dark theme toggle
* 🛍 Interactive product catalog
* 🛒 Cart drawer system
* ❤️ Wishlist functionality
* 🔍 Search overlay
* 🧾 Checkout interface
* 🎬 Preloader animation
* 📱 Fully responsive design
